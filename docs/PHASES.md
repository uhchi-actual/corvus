# Build Phases

## Phase 0 - Scaffold

Repo structure, Docker, CI, Pages workflow, WSL2 Docker disk cap guidance, and
`.env.example`.

## Phase 1 - Ingest And Schema

CSV and emulator paths first, then live ELM327. Load `data/schema.sql` and seed
sample logs.

## Phase 2 - SQL Core

Three showcase SQL queries plus session health score. This is the numeric
analysis centerpiece.

## Phase 3 - Agent

Huginn and Muninn LangGraph graph, provider-agnostic LLM environment, mandatory
disclaimer, and `agent_trace` on every path.

## Phase 4 - Power BI

Report against the database with deviation, DTC, and fuel-trim pages.

## Phase 5 - Static Dashboard

Next.js 15 dashboard, uhchi theme, and GitHub Pages static export.

## Phase 6 - Polish

README, screenshots, and resume bullets.
